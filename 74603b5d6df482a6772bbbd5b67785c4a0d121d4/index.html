      • Sélectionner Create resource
      • Search input Box: container registry
      • selectionner create
      • Créer un groupe
      • selectionner Create
      • Attendre la notification
      • Sélectionner Go to resource
      • Enregistre Login server pour votre adresse URL
      • Sélectionner Access keys dans le menu de gauche
      • Sélectionner Admin user Enable dans la colonne de droite
      • Enregister Username et le premier password

Registre de conteneurs

Créer un compte de storage pour les vidéos

  • https://portal.azure.com/
    • Selectionner Create a resource dans le menu de gauche
    • Sélectionner Home
    • Sélectionner More service en haut a droite
    • Recherches storage account
    • Sélectionner Create
    • Remplir le formulaire, nom: videostreaming
    • Sélectionner Review + create
    • Sélectionner Go to resource dans la notification
    • Sélectionner Access Keys dans le menu de gauche
    • Prendre en note Storage account name et une clé d’accès
    • Sélectionner Container dans le menu Data storage à gauche
    • Séclectionner +Container pour créer un nouveau conteneur de storage
    • Entrer un nom “video” et sélectionner Create
    • Sélectionner votre nouveau conteneur de storage
    • Sélectionner upload
    • Upload SampleVideo_1280x720_1mb.mp4

Installer la Machine virtuelle

  • Installer VirtualBox https:///www.virtualbox.org/wiki/Downloads
  • Installer Vagrant https://www.vagrantup.lcom/downloads.html
  • Créer un répertoire pour tester vagrant

Tester la machine virtuelle

# essayer les commandes suivantes

az
docker
docker-compose
kubectl
cd /vagrant # pour accéder au répertoire partagé avec votre répertoire local
ls

Problèmes avec Vagrant

  • Erreur de démarrage sur Mac: https://www.howtogeek.com/658047/how-to-fix-virtualboxs-“kernel-driver-not-installed-rc-1908-error/
# Vagrant was unable to mount VirtualBox shared folders. 
# Pas certain que ce soit nécessaire
vagrant ssh
cd /opt
sudo wget -c http://download.virtualbox.org/virtualbox/5.1.28/VBoxGuestAdditions_5.1.28.iso -O VBoxGuestAdditions_5.1.28.iso
sudo mount VBoxGuestAdditions_5.1.28.iso -o loop /mnt
sudo sh /mnt/VBoxLinuxAdditions.run
exit
